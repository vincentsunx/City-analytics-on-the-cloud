About this project

Aim for this project

Cloud-Based

Developed a Cloud-based solution that exploits a multitude of virtual machines across the UniMelb Research Cloud

Data Collection

Twitter API, AURIN is our data source, using these two platforms to provide reliable data for analysis for this project

Data Analysis

Data analysis is Inspecting, cleansing, transforming, and modeling data with the goal of discovering useful information, informing conclusions, and supporting decision-making.

Map Explaination:

This Map can give website visitors a general understanding of the population in each region. This allows users to have a general understanding of the number of people in the subsequent analysis. The division of this map is the regional division that we will analyze later.

The steps we followed:

1. Use ansible to create instances on Melbourne research cloud.
2. Use Twitter-API streaming to fetch real-time tweets in Australia, and fetch AURIN database then store all data to CouchDB.
3. Use CouchDB mapreduce and Pandas dataframe to analyze Twitter and AURAIN data to prove four scenarios.
4. Data visualization with Apex Chart and Javascript.
5. Combine the design and implementation process and all scenarios into a well-structured report.
